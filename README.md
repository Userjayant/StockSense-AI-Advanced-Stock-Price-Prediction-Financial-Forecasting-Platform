# 📌 Project Overview

**StockSense AI** is an advanced AI-powered stock market forecasting and analytics platform developed using **Machine Learning**, **Deep Learning**, and **Flask Web Framework**.

The system analyzes historical stock market data, generates technical indicators, trains multiple predictive models, and visualizes future stock price forecasts through an interactive professional dashboard.

This project combines:

* Financial analytics
* Machine learning
* Time-series forecasting
* Interactive web development
* Data visualization
* AI-driven prediction systems

# 🚀 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Userjayant/StockSense-AI-Advanced-Stock-Price-Prediction-Financial-Forecasting-Platform.git
cd StockSense-AI-Advanced-Stock-Price-Prediction-Financial-Forecasting-Platform
```

---

## 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

## 3️⃣ Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

---

## 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 5️⃣ Run Flask Application

```bash
python app.py
```

---

## 6️⃣ Open in Browser

```bash
http://127.0.0.1:5000
```
